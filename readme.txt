== Description ==

Adds parent comment ID meta box to a comment editing page. Use to make existing comments threaded.

== Installation ==

1. Upload `edit-parent-comment-id` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
